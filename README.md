## Overview

**Fine Dine Mobile Application** is a cross-platform React Native application developed as part of an autonomous waiter robot and smart restaurant system. The mobile app provides both **customer-side functionality** and an **admin panel**, enabling seamless interaction between customers, restaurant staff, and the waiter robot through cloud-based IoT integration.

The application is designed to work alongside the Fine Dine web application and serves as a core control and management platform within the robot-assisted dining ecosystem.

---

## Role in the Waiter Robot & IoT System

The mobile application plays a dual role in the system:

### üë§ Customer Side

### üõ†Ô∏è Admin Panel

## System Workflow (High-Level)

1. Customer places an order through the mobile application
2. Order data is stored and synced in Firebase
3. Autonomous waiter robot retrieves order information
4. Robot delivers food to the assigned table
5. Admin monitors and manages the process via the app
